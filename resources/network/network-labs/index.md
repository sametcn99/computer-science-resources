## All Resources

- **[Cisco Packet Tracer](https://www.netacad.com/learning-collections/cisco-packet-tracer)** <Badge type="tip" text="Free" />
  - Free network simulation tool from Cisco for practicing routing, switching, and network configuration.
- **[CTF Academy - Network Forensics](https://ctfacademy.github.io/network/index.htm)** <Badge type="tip" text="Free" />
  - Free network forensics challenges teaching protocol analysis and Wireshark packet inspection skills.
- **[Inventive HQ Firewall Rule Simulator](https://inventivehq.com/tools/network/firewall-rule-simulator)** <Badge type="tip" text="Free" />
  - Interactive browser-based firewall rule simulator to build, test, and analyze firewall rulesets.
- **[Kurose/Ross Wireshark Labs](https://gaia.cs.umass.edu/kurose_ross/wireshark.php)** <Badge type="tip" text="Free" />
  - Free hands-on Wireshark labs covering HTTP, DNS, TCP, UDP, IP, NAT, DHCP, and more protocols.
- **[Malware Traffic Analysis](https://www.malware-traffic-analysis.net/training-exercises.html)** <Badge type="tip" text="Free" />
  - Free PCAP-based traffic analysis exercises for learning network forensics and packet inspection.
- **[NetworkDirection Labs](https://networkdirection.net/labsandquizzes/labs/)** <Badge type="tip" text="Free" />
  - Free pre-built network labs for Cisco CML and VIRL covering routing, switching, and security topics.
- **[Cybrary](https://www.cybrary.it/practice-lab/network-segmentation)** <Badge type="warning" text="Freemium" />
  - Free virtual network labs covering segmentation, routing, and infrastructure skills with guided exercises.
- **[NetPilot](https://www.netpilot.io/network-lab-online)** <Badge type="warning" text="Freemium" />
  - Browser-based multi-vendor network simulator with real Cisco, Juniper, Arista, and Nokia CLIs.
