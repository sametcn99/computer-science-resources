## 🤝 Contributing

This is an open-source project. If you know a great resource that should be here, please contribute!

1. **Fork** the repository
2. **Add** your resource to `data/resources.json`
3. **Run** `bun run generate` to regenerate pages
4. **Submit** a pull request

Check our [GitHub repository](https://github.com/sametcn99/computer-science-resources) for guidelines.
